About clampGen

This project helps to calculate responsive dimensions using CSS clamp function.

By entering max, min and target values, users get a dynamic and responsive CSS clamp output.

Enter the value where the responsive will start in the maximum value range For example: 1100px, enter the breakpoint value in the minimum value range For example: 768px (This value can be the starting value of tablet screens.) And finally enter the target value you want to clamp, when you press the Generate button you will get a responsive value in the max and min value range.

!This project works in "px"!
